## Possible Network Values

<br />
<mark style={{background: "#f0f0f0"}}>BITCOIN_MAINNET</mark> <mark style={{background: "#f0f0f0"}}>ETHEREUM_MAINNET</mark> 

## Headers
| X-API-KEY | Accept|
|----------|----------|
| apiKey | application/json | 

### Body

```json copy filename="json"
{
    "network": "ETHEREUM_MAINNET"
}
```

## Sample Request
```js copy filename="Sample Request"
curl --location -g '{{baseUrl}}/api/wallets' \
--header 'X-API-KEY: {{apiKey}}' \
--header 'Accept: application/json' \
--data '{
    "network": "ETHEREUM_MAINNET"
}'
```
